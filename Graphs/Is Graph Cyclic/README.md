# Is Graph Cyclic

1. You are given a graph.
2. You are required to find and print if the graph is cyclic.

## Constraints
None

## Format
### Input
Input has been managed for you

### Output
true if the graph is cyclic, false otherwise

## Example
### Sample Input

7   
6   
0 1 10  
1 2 10  
2 3 10  
3 4 10  
4 5 10  
5 6 10

### Sample Output
false
