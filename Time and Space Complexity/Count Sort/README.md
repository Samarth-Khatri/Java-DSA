# Count Sort

1. You are given an array(arr) of integers.
2. You have to sort the given array in increasing order using count sort.

## Constraints
1 <= N <= 10000   
-10^8 <= arr[i] <= 10^8

## Format
### Input
An Integer n   
arr1  
arr2..  
n integers  

### Output
Check the sample ouput and question video.

## Example
### Sample Input

5 
7   
-2  
4   
1   
3   

### Sample Output
-2  
1   
3   
4   
7
