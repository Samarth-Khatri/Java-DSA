# Quick Select

1. You are given an array(arr) of integers.
2. You have to find the k-th smallest element in the given array using the quick-select algorithm.

## Constraints
1 <= N <= 100000  
-10^9 <= arr[i] <= 10^9  
1 <= k <= N

## Format
### Input
An Integer n   
arr1  
arr2..  
n integers  
An integer k

### Output
Check the sample output and question video.

## Example
### Sample Input

5  
7  
-2  
4  
1  
3  
3  

### Sample Output
pivot -> 3  
Swapping -2 and 7  
Swapping 1 and 7  
Swapping 3 and 4  
pivot index -> 2  
3
