# Sort 012

1. You are given an array(arr) containing only 0's, 1's, and 2's.
2. You have to sort the given array in increasing order and in linear time.

## Constraints
1 <= N <= 10000  
arr[i] = 0,1,2

## Format
### Input
An Integer N  
arr1  
arr2..  
n integers

### Output
Check the sample output and question video.
